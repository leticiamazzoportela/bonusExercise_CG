                                        Instruções

Compilação:
    gcc nomeAlgoritmo.c -lGL -lglut -lGLU

Execução:
    ./a.out

Exemplo:
    gcc bonusExercise_1.c -lGL -lglut -lGLU
    ./a.out

O programa inicia no modo "translação". Para
passar para o modo "rotação", tecle r, para
voltar para o modo "translação", tecle t.

Obs: Instruções para sistema Linux com GCC e OpenGL instalados.
